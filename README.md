# call the activity recorder to record the activity
# call the workflow planner to create workflow 
# call the execute workflow to execute the workflow

python activity_recorder.py
Copy ###WORKFLOW_RECORDING_START### in terminal and hit enter 
It will start recording the activity 

After workflow is completed
Copy ###WORKFLOW_RECORDING_STOP### in terminal and hit enter 
It will stop recording the activity 

python workflow_planner.py - it will create workflow 
python execute_workflow.py - it will execute the workflow
